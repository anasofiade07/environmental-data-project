Fossil Fuels Related Plots

Completion Date: April 3rd, 2024

Used Technologies: Python, HTML, CSS

Completion Date: April 3rd, 2024

Link to the project website Link to the project github

In this project, we dived into the time series analysis of fossil fuels, comparing and studying different components and elements within the fossil fuel industry. The analysis results were visually represented on graphs using a variety of programming languages such as HTML, CSS, and Python. Throughout this project, insights were gained into the trends and patterns fossil fuels, providing information and visual understanding of this crucial sector for us humans.

Hours Worked: 11 hours
